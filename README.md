# roll
#Finding local roller skating events and meeting other skaters can be surprisingly difficult. Information is often spread across #social media, group chats, and word of mouth, making it hard for beginners and experienced skaters alike to find a community. #We built RollCall to create one place where skaters can discover events, connect with others, and grow within the skating #community.
